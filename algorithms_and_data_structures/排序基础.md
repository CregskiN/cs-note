+ 选择排序法
  + 可以原地排序
  + 复杂度永远是O(n^2)

+ 插入排序法

  + 复杂度是O(n^2)，对于有序数组是O(n)

  ```java
  public static <E extends Comparable<E>> void sort2(E[] arr) {
    for (int i = 0; i < arr.length; i++) {
  		E tem = arr[i]; // 将 arr[i] 插入到合适位置
      int j;
      for(j = i; j - 1 >= 0 && tem.compareTo(arr[j - 1]) < 0) {
        // arr[j] < arr[j - 1] 将arr[j-1] 后移
        arr[j] = arr[j - 1];
      }
      arr[j] = tem;
    }
  }
  
  // 从后往前排
  public static <E extends Comparable<E>> void sort2(E[] arr) {
    for (int i = 0; i < arr.length; i++) {
  		E tem = arr[i];
      int j;
      for(j = i; j + 1 <= arr.length && tem.compareTo(arr[j + 1]) > 0; j ++){
        arr[j] = arr[j + 1];
      }
      arr[j] = tem;
    }
  }
  ```
