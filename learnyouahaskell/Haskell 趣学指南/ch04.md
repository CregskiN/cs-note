# ch03. 函数的语法



## 3.1 模式匹配

```haskell
-- 模式匹配
lucky :: (Integral a) => a -> String
lucky 7 = "LUCKY NUMBERS SEVEN."
lucky x = "Sorry, you are out of lucky."

GHCI> lucky 8
"Sorry, you are out of lucky."
GHCI> lucky 7
"LUCKY NUMBERS SEVEN."
GHCI> :t lucky
lucky :: Integral a => a -> String
```

1. 对输入数字非 1-5 作区分

   ```haskell
   sayMe :: (Integral a) => a -> String  
   sayMe 1 = "One!"  
   sayMe 2 = "Two!"  
   sayMe 3 = "Three!"  
   sayMe 4 = "Four!"  
   sayMe 5 = "Five!"  
   sayMe x = "Not between 1 and 5"
   
   GHCI> :t sayMe
   ```

