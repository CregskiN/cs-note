# Basic

## 1. 前置

### 1.1 clang/clang++

C源程序－> 预处理 －> 编译 －> 汇编 －> 链接程序 －> 可执行文件

```shell
# 一步到位，生成可执行文件
clang++ main.cpp
# 执行可执行文件
./a.out
```

一步到位，生成 `a.out` 文件



| 命令                        | 说明            |
| --------------------------- | --------------- |
| clang++ main.cpp -o myout.o | 生成myout.o文件 |
|                             |                 |
|                             |                 |
